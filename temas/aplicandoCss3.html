Aplicando CSS3 al Html5

Antes de comenzar les recomiendo utilizar Crome para visualizar resultados, porque como sabran Crome se actuliza solo y ya soporta css3 sin la necesidad de utilizar prefijos y sino actualizen sus navegadores ala ultima version para evitar ocuparar los prefijos -webkit-, -moz-, -o- y -ms- para versiones anteriores de los navegadores

CSS3 es la ultima version hasta la fecha y presenta como principales caracteristicas mayor control sobre el estilo de los elementos de nuestra pagina web y mayor numero de efectos visuales.Con esto estan pretendiendo darle muerta Flash y yo creo que si lo van a lograr.

Comenzemos con lo ultimo en que nos quedamos.Primero hay que agregar la hoja de estilo a nuestro documento html5.

Bueno Creamos una carpeta css y dentro de ella un documento llamado estilo.css. Lo abrimos y empezamos con lo basico

Apartir de aqui se empiezan a declarar las etiquetas utilizadas en el html5, para esto se utiliza un estandar donde todas las etiquetas van en orden alfabetico con sus correspondientes atributos.

Editamos body

Siguiendo el estandar Editamos Footer o pie de pagina

Editamos el h1 o el titulo

Editamos la lista dentro de nav

Editamos los elementos de la lista

Editamos el evento Hover para la lista

Y finalmente editamos el id contenido de la pagina
